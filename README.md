## ğŸ’» Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

- Ruby on Rails
- Rspec

<!--
Example command:

```sh
 gem install rails
```
 -->

### Setup


- Go to the terminal and `cd` into the directory of your preference.
- Run `git clone git@github.com:alphayowakarindi/njema.git` to clone the app.


### Install

- Run `bundle install` to install all the dependencies required to run the app.

### Usage

- Run `rails s` to start the app.

### Run tests

- For rails run `rubocop --color`
- For style run  `npx stylelint "**/*.{css,scss}"`
- For integration and unit tests run `rpec spec` to run all thes test files or `rspec spec/file-path` 
   to run a given file
